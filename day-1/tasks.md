## 3. Explain of types of AI on the basis of tasks.

**Predictive AI**  
Systems that infer unknown or future values from historical data. Formally, they learn a mapping $f: X \rightarrow Y$. Common uses include classification, regression, and forecasting.

**Generative AI**  
Systems that model data distributions and generate new samples. They approximate $P(X)$ or $P(X∣Y)$. Outputs include text, images, code, and audio.

**Agentic AI**  
Systems that act autonomously within an environment. They select actions over time to optimize a defined objective, often using feedback loops, planning, and tool interaction.
## 4. Explain of types of AI on the basis of era.

**Rule-Based Era (Symbolic AI)**  
Systems operate using explicitly defined rules. No learning from data; behavior is fully deterministic.

**Statistical / Machine Learning Era**  
Systems learn patterns from data using statistical methods. Enables probabilistic inference and generalization.

**Foundation Model Era**  
Large-scale models trained on diverse datasets. Capable of multi-task generalization with minimal task-specific tuning, often via prompting.

## 5. Explain Prompt and its types.

**Prompt (Definition)**  
A prompt is a structured input provided to an AI model to condition its output. It defines task constraints, context, and expected behavior.

**Types of Prompting**

- **Zero-shot**: No examples provided; relies on prior training.
- **One-shot**: Exactly one example included to guide output.
- **Few-shot**: Multiple examples included to establish patterns.
- **Role-based**: Assigns a role or perspective to constrain tone, structure, or reasoning style.
