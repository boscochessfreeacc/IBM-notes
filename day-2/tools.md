# AI TOOLS

## ChatGPT  

General-purpose AI assistant for text generation, coding, reasoning, and multimodal tasks.

## GitHub Copilot  

DE-integrated code completion and generation based on context.

## Descript  

Edits audio/video via text transcripts; includes voice cloning and screen recording.

## Gamma  

Generates structured presentations, docs, and webpages from prompts.
   
## Dialogflow  
 
Framework for building chatbots and voice assistants using intent-based NLP.
   
## Claude  
 
Large language model optimized for long-context reasoning and analysis.
  
## Gemini  
 
Multimodal model for text, code, and image understanding.
   
## Tabnine  

Predictive coding assistant with privacy-focused deployment options.
   
## Google AI Studio  

Interface for prototyping and testing Gemini models via prompts and APIs.
   
## Presentory 

AI presentation generator; similar category to Gamma.
   
## Perplexity AI 

Retrieval-augmented search delivering answers with citations.
  
## Antigravity 
 
No widely recognized AI product under this exact name; confirm source.
   
## Pictory  
 
Converts text or long-form content into short videos with visuals and captions.
   
## Codium  

Code generation and completion; often positioned as a Copilot alternative.
   
## Winston AI  

AI detection system designed to classify text as human- or AI-generated. Commonly used in academic and editorial workflows for plagiarism and authorship verification.

## ZeroGPT  

Estimates likelihood that text is AI-generated.
